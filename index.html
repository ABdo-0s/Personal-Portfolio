Smart System File Organizer

Project Overview:

A powerful system utility built to eliminate digital clutter by intelligently routing files into organized folders. By integrating core system operations into a sleek CustomTkinter GUI, it transforms a tedious terminal task into a high-speed, visually polished desktop application.

Some Features:

  • Instant Sorting: Cleans hundreds of messy files into neat folders in seconds.
  • Auto-Folders: Automatically detects and creates a home for every file type.
  • One-Click Fix: Organizes your entire desktop with a single button press.
  • Safe Migration: Uses reliable system logic to keep your data secure.

The Stack:

  • Python
  • CustomTKinter
  • os
  • shutil

YT StreamMaster

Project Overview:

A high-performance YouTube downloader engineered to automate playlist, video and audio batch downloading.
It transforms complex multi-threading, stream merging, and quality optimization into a seamless one-click solution, blending heavy-duty Python-powered backend processing with a sleek, modern CustomTkinter GUI.

Some Features:

  • 2K Resolution: Automated high-res merging via FFmpeg.
  • Bulk Processing: Queue multiple unique URLs for simultaneous extraction.
  • 4 Output Modes: Toggle between Audio, Video, Combined, or Separated.
  • Smart Alerts: Real-time toast notifications for status and errors.

The Stack:

  • Python
  • CustomTKinter
  • Multithreading
  • FFmpeg
